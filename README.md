# CyberLearn

CyberLearn is an interactive **cybersecurity learning platform** designed to help students and beginners strengthen their skills. It provides **quizzes, challenges, and educational resources** to make learning security concepts engaging and practical.

---

## 🚀 Features

* 🔑 **Authentication & secure user data** (hashed passwords, safe login)
* 📝 **Quizzes** on cybersecurity fundamentals
* 🛡️ **Challenges** for hands-on practice
* 📊 **Progress tracking** for learners
* 👨‍🏫 **Admin panel** for managing content

---

## 🛠️ Tech Stack

* **Frontend:** React.js
* **Backend:** Python (Flask/Django/FastAPI – to be chosen)
* **Database:** PostgreSQL / MySQL
* **Security:** User data hashing, authentication, input validation
